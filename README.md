# **Zero Point - Операцеонная Система**
**⚠️На данный момент Zero Point находится в активной стадии разработки** \

**⚠️ZeroPoint предназначена только для AArch64 и ARMv8. Поддержка ARMv7< и ниже, а так-же AArch32 не планируется** \
**💻ОС работает пока только в QEMU с использованием virt Cortex-a72. Но планируется запуск на Rasberry Pi5**
