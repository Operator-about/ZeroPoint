Большие изменения:
1. Добавление поддержки GICv2
2. Удаление Interrupts-common из-за создание более детальной настройки GICv2 и GICv3.
3. Создание отдельных файлов под UART, GICv2 и GICv3
4. Новая файловая структура проекта
5. Добавление L2 таблицы для GIC, UART и другой переферии в будущем
6. Автоматизированая активация прерываний(с 32 по 128) для GICv3/v2
7. Добавление Stringz.h
8. Переход с EL3 на EL2. Теперь ядро предназначено только для запуск от и в EL2 
9. Удаление kernel-modules.asm. Все функции были переписаны на C - см. в файл Kernel-modules.c
Средние изменения:
10. Добавление функций itos, stoi
11. Удаление функций Rx_clear и Tx_clear из UART.h/.c
12. Добавление поддержки 32-бит, 36-бит в IPS. 48-бит в IPS по прежнему остаются и являются более приоритетными чем 32, 36
13. Смена некоторых конструкций if-else на switch-case для оптимизации кода
14. Изменение AP атрибута у дескрипторов NORMAL типа под L1 таблицу
Мелкие изменения:
15. Удаление прослойки EL1h_configure_finish из-за неэффективности и ненадобности
16. ОС сразу-же после запуска переходит из EL2 NS в EL1h NS. Без прослоек main_EL2 и подобных
17. Удаление прослоек main_EL* 
18. Запуск GIC происходит только в EL1h NS
19. Умное заполнение L2 таблицы
20. Более правильный расчёт для GICv3 по структуре
